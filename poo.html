Programación Orientada a Objetos

Qué es la Programación Orientada a Objetos

La Programación Orientada a Objetos (POO) es un paradigma de programación basado en el uso de objetos. Los objetos combinan datos y comportamientos en una sola entidad. Los conceptos clave de la POO incluyen:

  • Clases: Plantillas para crear objetos. Definen atributos y métodos que los objetos pueden tener.
  • Objetos: Instancias de una clase. Contienen datos (atributos) y métodos (funciones) para operar sobre esos datos.
  • Herencia: Mecanismo que permite a una clase heredar atributos y métodos de otra clase.
  • Polimorfismo: Permite a las funciones utilizar diferentes implementaciones basadas en el tipo de objeto.
  • Encapsulación: Oculta los detalles internos del objeto y sólo expone lo necesario a través de métodos públicos.

Ejemplo Básico

A continuación, se muestra un ejemplo simple de una clase en pseudocódigo:

clase Animal {
    atributo nombre

    función constructor(nombre) {
        this.nombre = nombre
    }

    función hacerSonido() {
        imprimir("Sonido genérico")
    }
}

clase Perro hereda de Animal {
    función hacerSonido() {
        imprimir("Guau!")
    }
}

miPerro = nuevo Perro("Rex")
miPerro.hacerSonido() // Imprime "Guau!"
