Estructuras de Datos y Algoritmos Básicos

Introducción a Estructuras de Datos

Las estructuras de datos son formas de organizar y almacenar datos para que puedan ser utilizados de manera eficiente. Algunas estructuras de datos básicas incluyen:

  • Listas: Secuencias ordenadas de elementos. Los elementos se pueden añadir, eliminar o acceder por su índice.
  • Conjuntos: Colecciones de elementos únicos, sin un orden específico.
  • Diccionarios: Colecciones de pares clave-valor, donde cada clave es única y se asocia a un valor.

Algoritmos Básicos

Los algoritmos son secuencias de pasos para resolver problemas o realizar tareas. Algunos algoritmos básicos incluyen:

  • Búsqueda Lineal: Busca un elemento en una lista revisando cada elemento uno por uno.
  • Búsqueda Binaria: Busca un elemento en una lista ordenada dividiendo el espacio de búsqueda en mitades.
  • Ordenación por Inserción: Ordena una lista construyendo la lista ordenada elemento por elemento.

Ejemplo Básico

A continuación, se muestra un ejemplo simple de un algoritmo de búsqueda lineal en pseudocódigo:

función buscarElemento(lista, objetivo) {
    para cada elemento en lista {
        si (elemento == objetivo) {
            retornar índice del elemento
        }
    }
    retornar -1 // Elemento no encontrado
}
