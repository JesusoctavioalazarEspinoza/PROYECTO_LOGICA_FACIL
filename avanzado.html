Ejemplo Avanzado

A continuación, se muestra un ejemplo básico de un algoritmo de ordenación por mezcla (Merge Sort) en pseudocódigo:

función mergeSort(lista) {
    si (lista.tamaño <= 1) {
        retornar lista
    }

    medio = lista.tamaño / 2
    izquierda = lista.subLista(0, medio)
    derecha = lista.subLista(medio, lista.tamaño)

    izquierda = mergeSort(izquierda)
    derecha = mergeSort(derecha)

    retornar merge(izquierda, derecha)
}

función merge(izquierda, derecha) {
    listaOrdenada = nueva Lista()
    mientras (izquierda.noEstáVacía() y derecha.noEstáVacía()) {
        si (izquierda.primerElemento() < derecha.primerElemento()) {
            listaOrdenada.agregar(izquierda.quitarPrimerElemento())
        } sino {
            listaOrdenada.agregar(derecha.quitarPrimerElemento())
        }
    }
    listaOrdenada.agregarTodos(izquierda)
    listaOrdenada.agregarTodos(derecha)
    retornar listaOrdenada
}
