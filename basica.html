Estructuras Básicas

Variables y Tipos de Datos

Las variables son espacios de almacenamiento en memoria que se utilizan para guardar datos. Los tipos de datos determinan el tipo de valor que una variable puede almacenar. Los tipos de datos comunes incluyen:

  • Enteros: Números enteros (por ejemplo, 1, 2, 3).
  • Flotantes: Números decimales (por ejemplo, 1.5, 2.75).
  • Strings: Cadenas de texto (por ejemplo, "Hola Mundo").
  • Booleanos: Valores de verdadero o falso (true/false).

Operadores Básicos

Los operadores son símbolos que se utilizan para realizar operaciones en variables y valores. Algunos operadores básicos incluyen:

  • Operadores Aritméticos: + (suma), - (resta), * (multiplicación), / (división).
  • Operadores Relacionales: == (igual a), != (diferente de), > (mayor que), < (menor que).
  • Operadores Lógicos: && (y lógico), || (o lógico), ! (negación).

Estructuras Condicionales

Las estructuras condicionales permiten ejecutar diferentes bloques de código dependiendo de una condición. Las principales estructuras condicionales son:

  • if: Ejecuta un bloque de código si una condición es verdadera.
  • else: Ejecuta un bloque de código si la condición del if es falsa.
  • switch: Permite seleccionar entre múltiples opciones basadas en el valor de una variable.

Ejemplo Básico

A continuación, se muestra un ejemplo simple de una estructura condicional en pseudocódigo:

if (edad >= 18) {
    imprimir("Eres mayor de edad.");
} else {
    imprimir("Eres menor de edad.");
}
